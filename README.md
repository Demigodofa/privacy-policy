# privacy-policy

Public policy source for Material Guardian store and in-app privacy disclosures.

## Pages

- `index.html`
  - canonical public privacy policy page for App Store Connect and Google Play
- `terms.html`
  - public terms-of-use page for App Store subscription review and in-app legal links
- `delete-account.html`
  - public account-deletion request page for Google Play's required web deletion link
  - also usable as the optional Apple `User Privacy Choices URL`
